# Student Leaderboard

An interactive and responsive student leaderboard built with HTML, CSS, and jQuery. This project allows users to view and search through student rankings in a clean, user-friendly interface. 

## Features

- **Responsive Design**: Works seamlessly on both desktop and mobile devices.
- **Search Functionality**: Instantly filter leaderboard entries by typing in the search bar.
- **Scroll Support**: Smooth horizontal scrolling to ensure all content is accessible, even on smaller screens.
- **User-Friendly Interface**: Clean design with easy-to-read tables and smooth interactions.

## How to Use

1. Clone or download the project files to your local machine.
2. Open the `index.html` file in your preferred web browser.
3. Use the search bar to filter students by their names.
